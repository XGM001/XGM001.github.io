博客的通知公告
推存电脑端全屏访问，手机端访问无法显示搜索功能
如果遇到需要密码的博客文章，博客状态有几种状态：
1.博客正在完善，或者有部分不对的地方需要修改，
2.博客完善，但是付出了一定的代价，所以闭源。
如果查看打开加密文章，请通过邮箱联系本人 xgmcreate@foxmail.com
博客资料源如果涉及到版权问题，请联系作者，可以将资料源删除或者闭源，或者添加版权信息。

Notification announcements for blogs
The full-screen access on the computer side is pushed, and the search function cannot be displayed on the mobile terminal side.
If you encounter a blog post that requires a password, the blog status has several states:
1.The blog is being improved, or there are some wrong places that need to be revised,
2.The blog is perfect, but it pays a certain price, so it is closed source.
If you want to view the open encrypted article, please contact me via email xgmcreate@foxmail.com
If the blog data source involves copyright issues, please contact the author, you can delete or close the source, or add copyright information.
